# 🧠 Match Arena

Jogo da memória moderno com sistema de pontuação, combo, ranking local e modo contra o tempo.

🔗 **Jogar agora:**  
https://danilo-zanatta.github.io/match-arena/

---

## 🎮 Funcionalidades

- Modo clássico
- Modo contra o tempo
- Dificuldades: fácil, médio e difícil
- Sistema de pontuação:
  - +5 por acerto
  - -1 por erro
- Sistema de combo com bônus
- Ranking local (Top 10)
- Música de fundo
- Interface responsiva (mobile, tablet e desktop)

---

## 📱 Experiência

O jogo foi projetado com abordagem **mobile-first**, garantindo:

- Todos os cards visíveis no celular
- Interface adaptada para toque
- Layout otimizado para diferentes telas

---

## 🛠️ Tecnologias utilizadas

- HTML5

- JavaScript (vanilla)
- GitHub Pages (deploy)
